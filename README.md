# Distribuciones_Probabilidad

Repositorio con ejemplos de distribuciones de probabilidad (discretas y continuas) en Python.

## Contenido
- `distribucion_binomial.py` : Ejemplo de la distribución binomial (cálculo manual vs scipy).
- `distribucion_poisson.py` : Ejemplo de la distribución de Poisson.
- `distribucion_normal.py` : Ejemplo de la distribución normal (CDF manual vs scipy).
- `distribucion_exponencial.py` : Ejemplo de la distribución exponencial.
- `aplicacion_empresarial.py` : Simulación Monte Carlo de demanda diaria (Poisson) para análisis de stockout e ingresos.

## Dependencias
Recomendado crear un entorno virtual e instalar:
```
pip install numpy scipy matplotlib seaborn pandas
```

## Uso
Ejecuta cada script con:
```
python distribucion_binomial.py
```
Los scripts mostrarán las figuras al ejecutarse y no guardan archivos de imágenes.
